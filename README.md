# mypy quick example

- Clone the repository
- Create a virtual environment with Python 3
    ```
    virtualenv venv
    source venv/bin/activate
    ```

- Install the packages
    ```
    pip install -r requirements.txt
    ```

- Run mypy to perform the static checking
    ```
    mypy main.py
    ```

- Run the main script
    ```
    python main.py
    ```
